# Reference Thumbnails

This folder is for uploading reference thumbnails from your previous successful videos.

## Purpose

- Store high-performing thumbnail examples from your past videos
- Help the thumbnail generator agent understand your visual style and branding
- Provide inspiration and reference points for new thumbnail designs

## Usage

1. Upload your best-performing thumbnails here
2. Name them descriptively (e.g., `high_ctr_gemini_video.png`, `viral_agent_tutorial.png`)
3. The agent can reference these when generating new thumbnail concepts

## Recommended Thumbnails to Include

- Thumbnails with high CTR (Click-Through Rate)
- Thumbnails that drove significant engagement
- Different styles you've experimented with
- Thumbnails representing different video categories on your channel

## File Format

- Recommended: PNG or JPG
- Aspect Ratio: 16:9 (YouTube standard: 1280x720 pixels)
